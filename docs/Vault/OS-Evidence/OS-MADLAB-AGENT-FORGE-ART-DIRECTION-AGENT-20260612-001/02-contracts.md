1. **art_direction_brief.schema.json:** Schema de entrada contendo objetivos narrativos, conceituais e referências.

3. **scene_card.schema.json:** Estrutura para descrever atmosfera, iluminação e restrições de movimento da cena.
4. **visual_world_bible.schema.json:** Schema global para mundos visuais por projeto.
