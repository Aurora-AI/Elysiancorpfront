00 — Narrative Architecture
O sistema tem dois mundos deliberados. O Hero vive no Mundo Escuro — cinema alquímico, ouro e obsidiana. As seções de conteúdo transitam para o Mundo Claro — editorial, pergaminho e safira. Esta não é inconsistência: é o arco narrativo do produto. A marca revela uma segunda dimensão ao longo do scroll.

Lógica do Acento Duplo

O Hero usa Ouro (#D4AF37) — é o assinatura alquímica, intocável. O acento ouro emerge da textura, do gradiente de letra, do metadata.

As seções claras usam Safira (#1B4DA0 / #4A7FD4) — o acento do mundo revelado. A transição Hero → Seções é a transição Ouro → Safira.

O Teste de Autoridade (Manual da Agência)

"Se eu remover o efeito, a narrativa ainda existe?"

Se não — o efeito é a narrativa. Isso é o que queremos.
Se sim — o efeito é decoração. Não entra.

"Se o design precisa do fundo escuro para parecer premium, ele falhou."

Animação Engine Parâmetros GSAP Observação
Hero — Rule of Silence GSAP opacity:0→1, scale:1.1→1, duration:1.2s, delay:0.8s, ease:power4.out Delay obrigatório. Nunca animar imediatamente no load.
Headline chars (split por letra) GSAP ScrollTrigger yPercent:100→0, rotateX:±45→0, opacity:0→1, duration:1.5–1.8s, stagger:0.05s, ease:power4.out Trigger: top 60%. Chars individuais via .char class.
Imagem light (Answer) GSAP ScrollTrigger opacity:0→1, scale:0.9→1, duration:2s, ease:expo.out Overlap com headline: -=1.5
Imagem dark (Problem) GSAP ScrollTrigger clipPath: inset(0 100% 0 0)→inset(0 0% 0 0), scale:1.2→1, duration:2s, ease:expo.inOut Clip reveal cinematográfico. Apenas em seções escuras.
Parallax de imagem GSAP ScrollTrigger yPercent:±15–20, ease:none, scrub:true Start: top bottom → End: bottom top. Oposto à direção do scroll.
Blocos de conteúdo GSAP ScrollTrigger opacity:0→1, y:30–40→0, duration:1.2–1.5s, ease:power3.out Overlap com headline: -=1 a -=1.2
Header entrance GSAP opacity:0→1, y:-20→0, duration:1.5s, delay:1.2s, ease:expo.out Delay: logo após o Hero background iniciar reveal.
Botão hover CSS Transition background + color, 700ms, cubic-bezier(0.23,1,0.32,1) CSS puro — não precisa de GSAP. Sem transform. Sem sombra.
Hero — Abyss Tide (scroll) GSAP ScrollTrigger clipPath: inset(100% 0 0 0)→inset(0% 0 0 0), opacity:0→1, scrub:true Start: 10% top. mask: gradient to bottom. Exclusivo do Hero — não replicar.
Hero — stop-motion flicker setInterval 80ms normal · 320ms onHover Sequência alternada: letra → retrato, para cada char de ELYSIAN. Exclusivo do Hero.
Performance obrigatória: Animar exclusivamente via transform e opacity. Nunca top, left, width, height. Meta: 60fps em mid-range (iPhone 13, Android 4GB RAM). Fallback CSS para todos os efeitos GSAP.

08 — Anti-Patterns (Banidos)
Estes padrões são proibidos. Cada um deles destrói a coerência do sistema ou contradiz a identidade Fluid Editorial da ElysianCorp.
Qualquer acento verde ou esmeralda — banido em todas as superfícies
Gold (#D4AF37) fora do Hero e seus componentes diretos
Pure black #000000 — usar Obsidian #0A0A0B
CRT scanlines, glitch retrô, efeito de cintilação na UI de seções
Industrial grid ou dot-matrix como background
Terminal logs, telemetria ou overlays de diagnóstico visíveis ao usuário
Copy com underscore: Integrity_Rate → Integrity Rate
Terminologia tática: FACTORY_ID, FORENSIC, VULNERABILITY_DETECTED, CLUSTER
Font Inter em qualquer contexto do produto
DM Mono fora de versões, IDs e coordenadas (jamais em body ou nav)
border-radius em elementos estruturais — somente 0
Outer glows, neon box-shadows, blur blobs de cor
Hero centrado — sempre assimétrico ou split
Scroll indicators: "Scroll to explore", chevrons, setas animadas
3 colunas iguais para features — usar zig-zag ou assimétrico
Elementos sobrepostos — todo elemento tem sua própria zona espacial
h-screen — usar min-h-[100dvh] (bug crítico em iOS Safari)
Gradiente de texto em titulares de seção — somente no Hero (via webkit-clip)
Clichê AI: "Elevate", "Seamless", "Unleash", "Next-Gen", "Revolutionize"
Números redondos falsos: 99.99%, 10x faster, 50% ROI sem contexto
