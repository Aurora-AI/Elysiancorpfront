# Sandeep Swadia (MIT Monk) - Part 2: Range & Wicked Environments

## Core Tesis
In a "Wicked World" where AI automates entry-level cognitive work, super-specialization is a risk. Resilience comes from "M-Shape" architectures: multiple deep pillars connected by an invisible thread.

## The L.I.N.K. Framework (Domain Audit)
1. **L (Language)**: Does Domain A use the same language/logic as Domain B?
2. **I (Improve)**: Does getting better at A improve B?
3. **N (Need)**: Do you need tools from A to solve B?
4. **C (Cut)**: If you cut A, does B suffer?
*Score 3+ indicates composition. Score < 2 indicates distraction.*

## The Sorting Hat (4 Jobs of Interest)
1. **Mastery**: Skills to be the best (Core).
2. **Stability**: Skills that buy today's bills.
3. **Optionality**: Skills that buy tomorrow's pivots.
4. **Restoration**: Skills that restore mental energy.

## Rules
- **Wicked Survival**: Range wins over early specialization in dynamic markets.
- **M-Shape**: Multiple pillars are the "safety rope" against the "sinking floor" of AI automation.
- **Uncarved Block**: Refuse to be defined by a single tool or role.
